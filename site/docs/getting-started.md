# Getting started

## Install Typedex

::: info
The library hasn't been published on NPM yet, so feel free to install it from Git.
:::

::: code-group

```sh [npm]
npm install @typedex/indexed-db
```

```sh [yarn]
yarn add @typedex/indexed-db
```

```sh [pnpm]
pnpm add @typedex/indexed-db
```

```sh [bun]
bun add @typedex/indexed-db
```

```sh [deno]
deno add npm:@typedex/indexed-db
```

:::
